The `nest` sensor platform let you monitor sensors connected to your [Nest](https://nest.com) thermostat and/or your Nest Protect Smoke Alarm.

To set it up, add the following information to your `configuration.yaml` file:

```yaml
sensor:
  platform: nest
  monitored_conditions:
    - 'temperature'
    - 'target'
    - 'humidity'
    - 'mode'
    - 'last_ip'
    - 'local_ip'
    - 'last_connection'
    - 'battery_level'
    - 'weather_condition'
    - 'weather_temperature'
    - 'weather_humidity'
    - 'wind_speed'
    - 'wind_direction'
    - 'co_status'
    - 'smoke_status'
```

Configuration variables:

- **monitored_conditions** array (*Required*): States to monitor.
  - 'temperature'
  - 'target'
  - 'humidity'
  - 'mode'
  - 'last_ip'
  - 'local_ip'
  - 'last_connection'
  - 'battery_level'
  - 'weather_condition'
  - 'weather_temperature'
  - 'weather_humidity'
  - 'wind_speed'
  - 'wind_direction'
  - 'co_status' # Nest Protect only
  - 'smoke_status' # Nest Protect only

<p class='note'>
You must have the [Nest component](/components/nest/) configured to use this sensor.
</p>
